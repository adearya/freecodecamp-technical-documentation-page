Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language.

JavaScript contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements.

What you should already know

Before starting to learn JavaScript, you should have a basic understanding of HTML and CSS.

JavaScript and Java

JavaScript is similar to Java in some ways but fundamentally different in others. JavaScript is a loosely typed language.

Hello world

A simple JavaScript code example to output "Hello, World!" to the console is shown below.

console.log("Hello, World!");
Variables

Variables are containers for storing data values.

Declaring variables

In JavaScript, you can declare variables using var, let, or const.

Variable scope

Variable scope refers to the accessibility of variables within a program.

Global variables

Global variables are declared outside any function and can be accessed from any part of the code.

Constants

Constants are variables that, once assigned a value, cannot be reassigned.

Data types

JavaScript supports various data types, including strings, numbers, booleans, objects, and undefined.

if...else statement

The if...else statement is used to perform different actions based on different conditions.

while statement

The while statement creates a loop that executes as long as a specified condition is true.

Function declarations

A function declaration defines a named function in JavaScript.
